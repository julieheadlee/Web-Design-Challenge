    • Summary: Latitude vs. X

    The purpose of this analysis was to examine how latitude affects various weather measurements: maximum temperature, humidity, cloudiness, and wind speed. Specifically we asked how weather changes as you get closer to the equator. By analyzing data from the OpenWeatherMap API based on over 500 cities, using Matplotlib, we could determine trends and correlations between latitude and weather.
